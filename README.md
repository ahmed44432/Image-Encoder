# 🖼️ Image Encoder

A modern Windows Forms desktop application built with **.NET 10** and **C#**. This application is designed to securely encrypt and decrypt images to protect user privacy using dedicated cryptographic libraries.

## ✨ Key Features

* 🔒 **Secure Image Encryption:** Encrypt your private image files so they cannot be viewed or accessed through traditional image viewers.
* 🔓 **Fast Decryption:** Easily restore encrypted files back to their original state with zero quality loss.
* 🎨 **Modern UI/UX:** Designed using the `ReaLTaiizor` library to deliver a sleek, user-friendly, and dark-themed visual experience.
* ⚡ **Optimized Performance:** Leverages the latest capabilities of .NET 10 for fast file processing and smooth execution.

---

## 🛠️ Built With

* **Language:** [C#](https://learn.microsoft.com/en-us/dotnet/csharp/)
* **Framework:** .NET 10.0 (Windows Forms)
* **UI Customization:** `ReaLTaiizor` (for advanced UI components, custom buttons, and themes)
* **Core Logic:** `SimpleCrypt` (handling the image encryption and decryption algorithms)
